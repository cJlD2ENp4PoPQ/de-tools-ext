Die-Ewigen Extension

Speicher zurücksetzen

Hier kannst du die gespeicherten Erweiterungsdaten manuell zurücksetzen. Das Zurücksetzen kann nicht rückgängig gemacht werden.

Alle Daten zurücksetzen Setzt sämtliche Erweiterungsdaten für alle Server auf den Ausgangszustand zurück.

Daten nach Server

Jeder Spielserver speichert seine Daten getrennt. Hier kannst du Daten einzelner Server oder einzelner Kategorien gezielt zurücksetzen.
